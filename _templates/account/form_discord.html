UpKoding memiliki server Discord sebagai tempat kamu nongkrong dan berdiskusi dengan teman-teman pengguna upkoding lainnya.

Namun sebelum bisa aktif di Discord, kamu wajib membuktikan bahwa kamu memiliki akun di upkoding.com dengan melakukkan verifikasi, caranya:

  1. Gabung di server Discord UpKoding: https://discord.gg/yaxjhTrFGf
  2. DM UpKodingBot dengan command /verifikasi dan masukkan token dibawah.
  3. Apabila verifikasi berhasil, kamu langsung diberikan akses untuk berdiskusi.
Token dibawah sifatnya rahasia dan hanya bisa dipakai sekali, apabila orang lain menggunakan token dibawah sebelum kamu maka orang itulah yang akan mendapat akses ke Discord.
