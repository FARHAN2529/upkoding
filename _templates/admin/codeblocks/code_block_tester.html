Challenge tips

Dalam membuat code challenge, bisa dipertimbangkan hal berikut:

  • Correctness test, bisa menggunakan assert untuk mengecek nilai sebuah variable benar atau output sebuah fungsi sudah sesuai.
  • Output test, gunakan expected output untuk mengecek apakah output program benar.
  • Perhatikan apakah challenge yang kita buat bisa di bypass oleh user apa tidak.
  • Pada saat membuat codeblock, tentukan mana yang tidak boleh diedit user (readonly) dan mana yang tidak.
